# 📖 RetailMart India Business Analytics Case Study

## 📌 Project Summary

RetailMart India is a retail sales analytics project developed to help business stakeholders monitor sales performance, customer behavior, regional performance, and product trends using interactive dashboards.

The project was built using **MySQL**, **Power BI**, **Power Query**, and **DAX** to transform raw sales data into meaningful business insights.

---

# 🎯 Business Objective

Retail companies generate thousands of transactions every day. The primary objective of this project is to transform raw sales data into an interactive dashboard that enables decision-makers to:

- Monitor overall business performance
- Track sales and profit growth
- Analyze customer purchasing behavior
- Identify top-performing products and regions
- Support data-driven business decisions

# 📂 Dataset Overview

The project uses three datasets:

### Orders
Contains transaction-level sales information including:
- Order ID
- Customer Details
- Product Details
- Sales
- Profit
- Quantity
- Discount
- State
- Region

### Returns
Contains returned orders used for business analysis.

### People
Contains regional manager information.
